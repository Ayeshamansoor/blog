What is Javacript?

JavaScript is a lightweight, interpreted programming language. It is designed for creating network-centric applications. It is complimentary to and integrated with Java. JavaScript is very easy to implement because it is integrated with HTML. It is open and cross-platform.

JavaScript is a platform-independent language. Any browser supporting JavaScript can run the code irrespective of the operating system JavaScript uses the just-in-time compilation technique. Since the compilation is handled at run time, JavaScript is considered an interpreted language.
